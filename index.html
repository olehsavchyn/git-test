  • git clone <path-to-repository> - клонировать репозиторий на ПК....
  • git status - статус текущего локального репозитория
  • git add <path-to-file> - добавить новый файл под контроль гита
  • git commit -am "added new file index.html" - зафиксировать изминения и дать им коментарий
  • git push - залить все закомиченые изминения в репозиторий на сервер
  • git pull - забрать все последние обновление к себе на ПК
